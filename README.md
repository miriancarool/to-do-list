# To Do List

Aplicação web simples para gerenciamento de tarefas

Tecnologias usadas: HTML5, CSS3 e JavaScript

Esse gerenciador de projetos permite que o usuário escreva todas as tarefas a serem feitas para se organizar, podendo editar a tarefa e depois também excluir.

O objetivo é praticar a manipulação do DOM e usar logica de programação front-end.
